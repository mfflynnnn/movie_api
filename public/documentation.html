This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
List of all movies /movies GET movies None A JSON object holding data about all the movies
Data about a single movie by title /movies/[title] GET title None A JSON object holding data about a single movie by title. Example: { title: "Inside Out", director: "Pete Docter", genre: "family animated", image-URL: "https://en.wikipedia.org/wiki/Inside_Out_%282015_film%29#/media/File:Inside_Out_(2015_film)_poster.jpg", featured-film: "Yes"}
Data about a genre by genre name (e.g., “Thriller”) /movies/genres/[genre] GET genre None A JSON object holding data about a genre by genre name. Allow new users to register with a username /users POST users A JSON object with the user's selected username, structured like: {username: 'jenique1234'} A text message indicating if the username was successfully registered
Allow users to update their username /users/[username] PUT username A JSON object with the user's updated username, structured like: {username: 'jenique4321'} A text message indicating if the username was successfully updated
Allow users to add a movie to their list of favorites /movies/favorites PUSH favorites A JSON object with the favorited movie, structured like: {title: 'Inside Out'} A text message indicating that the movie was added to the list of favorites
Allow users to remove a movie from their list of favorites /movies/favorites/[title] DELETE title A JSON obect with the movie to be removed, structured like: {title: 'Eternal Sunshine of the Spotless Mind'} A text message indicating that the movie was removed from the list of favorites
Allow existing users to deregister /users/[username] DELETE username A JSON object with the username to be deregistered, structured like: {username: 'jenique4321'} A text message indicating that the username was deregistered
